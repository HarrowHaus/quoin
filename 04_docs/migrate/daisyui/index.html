Quoin Spec Playground Packs Get started GitHub Home / Migrate / From DaisyUI Migrate from DaisyUI DaisyUI proved that semantic compression on top of Tailwind works. Quoin extends the same compression one layer higher — from class names to elements. If you like DaisyUI, you'll find Quoin familiar; the move is mostly a syntax shift. DaisyUI is the prior art that validated this whole direction. Quoin's vocabulary borrows the same semantic instinct (alert, badge, card, divider) and lifts it from class attributes into element names. The shape of the change

DaisyUI ships a Tailwind plugin that adds semantic class names like btn-primary, alert alert-warning, card card-bordered. Quoin lifts those into HTML element names, with the same naming convention preserved in @quoin/vocab-daisy.

<!-- DaisyUI --> <button class="btn btn-primary">Save</button> <div role="alert" class="alert alert-warning"> Build will run with reduced cache. </div> <div class="card card-bordered bg-base-100"> <div class="card-body"> <h2 class="card-title">Key takeaway</h2> <p>The aesthetic is the swappable vendor.</p> </div> </div> <!-- Quoin --> <primary-action>Save</primary-action> <alert-tile intent="warning"> Build will run with reduced cache. </alert-tile> <card-bordered> <stack> <lead-graf weight="emphasize">Key takeaway</lead-graf> <reading-flow> <p>The aesthetic is the swappable vendor.</p> </reading-flow> </stack> </card-bordered> Translation table

Recommended pack stack npm install @quoin/tokens-baseline @quoin/vocab-daisy @quoin/vocab-editorial @quoin/impl-tailwind

vocab-daisy adds the alert / badge / card / divider / indicator / modal primitives that match the DaisyUI instinct. vocab-editorial covers the long-form reading primitives DaisyUI doesn't ship. impl-tailwind emits Tailwind v4 classes you already know.

Theming

DaisyUI ships ~35 named themes via CSS variables. Quoin's equivalent is the token-pack mechanism: pick any of the forty harvested packs or supply a project-local quoin.tokens.json override. Same CSS-variable approach, broader pack ecosystem.
