    A semantic design language with a build-time compiler. You write intent-bearing elements; the compiler resolves them against pluggable packs into standard HTML and CSS.

    Install @quoin/compiler, pick one pack of each type (token, vocab, impl), wire the Vite plugin. Five minutes for anyone who's used Tailwind. See the getting started guide.
