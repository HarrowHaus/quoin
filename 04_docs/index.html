The compiler renders them to standard HTML and CSS at build time, using a pluggable token pack to determine the aesthetic. The browser sees no Quoin-specific runtime. Get started Open the playground Read the spec The four layers

The page you're reading was written with Quoin semantic elements. Below the semantic layer, three other layers do the work the author never has to repeat.

Authoring Semantic web-component-style tags. <authority-mark>, <reading-flow>, <primary-action>. The vocabulary lives in a vocabulary pack. Tokens Three-layer DTCG tokens — base palette, semantic namespace, component overrides. The aesthetic lives in a token pack. Compilation Build-time. Resolves semantic tags against the active packs and emits standard HTML + CSS. Zero runtime dependency in the output. Rendering The browser. Receives plain HTML and CSS. No proprietary engine, no shadow DOM, no Quoin-specific runtime. A button, three ways

Intent is the durable contract. Intellectual lineage

Christopher Alexander's A Pattern Language for the generative-grammar model. The Swiss / International Typographic Style and the brutalist editorial tradition for the naming and structural philosophy. DaisyUI for the prior-art proof that semantic compression works on top of Tailwind. Tailwind v4 for the token architecture. The W3C Design Tokens Community Group for the file format. Web Components for the optional runtime fallback path.
