The compiler runs entirely client-side here. In a real project the same compiler runs at build time through @quoin/compiler/vite and emits static HTML — no JavaScript needed at runtime to render Quoin output.

The token-pack picker covers all 30 packs Quoin ships in v0.1. The tier badge — Tier A, B, or C — indicates how the pack was extracted (byte-faithful, mapping notes, or designed approximation pending). See the pack browser for filtering across all packs by tier, license, and source system.

Use the Share link button to copy a URL that reproduces the current source + active packs. The state is encoded in the URL hash, so a paste in a new tab reconstructs everything.
