What just happened

The browser ran @quoin/compiler four times on the same source — once per active token pack. Each compilation produced its own HTML + CSS pair, dropped into a separate iframe alongside the matching token pack's custom-property stylesheet. The markup the author wrote is identical across all four; only the visual layer changed.

The same compiler runs at build time in a real Quoin project. There's no JavaScript runtime in the output — the page above just uses the browser entry to make the swap interactive for demonstration.
