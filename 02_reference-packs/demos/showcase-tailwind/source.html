The same compiler, the same token pack, no boundary between the two.

A Quoin build loads one or more vocabulary packs. The compiler unions their primitive sets and resolves source tags against the combined registry. Name collisions emit warnings and resolve last loaded wins — which means a project can layer a custom vocabulary pack on top of a stock one to override specific primitives without forking.

This demo loads vocab-editorial and vocab-dashboard together — 36 primitives in one source.
