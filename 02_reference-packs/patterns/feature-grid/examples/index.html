Cell-level microstates

Per-cell microstates demonstrated with data-mockstate for static rendering. In production these resolve via :hover, :focus-visible, [aria-selected], [aria-disabled], [aria-busy]. Interactive cells (those with data-interactive="true") are keyboard-tabbable and Enter/Space activates them.

    Default microstate. Card is tabbable and opens the destination on Enter.

  • Hover — shadow lift

    Shadow elevates, transform-Y -1px, border emphasises.

  • Selected — accent ring

    Accent-coloured border + 1px outer ring. Used in comparison-grids.

  • Disabled — pending review

    40% opacity, pointer-events none. Feature hidden from public view.

Accessibility checklist

  • Each cell is a semantic <article> with <h3>/<h4> heading; heading-level chosen by surrounding section context.
  • Leading icons are aria-hidden="true" when decorative; if the icon is the sole carrier of meaning, the article gets an explicit aria-label.
  • Numeral leading uses font-variant-numeric: tabular-nums so step numbers don't shift the cell width when they reach two digits.
  • Interactive cards (data-interactive="true") are tabindex="0"; activate on Enter or Space; focus ring honours --focus-ring-width + --focus-ring-offset tokens via :focus-visible.
  • Loading state: aria-busy="true" on the article + skeleton animation respects prefers-reduced-motion.
  • Error state: role="alert" on the grid wrapper so screen readers announce immediately.
  • Bento centerpiece's display-tier heading uses text-wrap: balance for typographically clean line breaks; max-width: 18ch keeps the headline pull-quote-like.
  • Steps layout's numbered cells use mono numerals; the visual sequence is reinforced by tab order and heading content.

Typography precision

Headings use the active --font-display; body uses --font-sans; code spans use --font-mono; the editorial glyph variant uses --font-serif. Curly apostrophes throughout. Em dashes for ranges. Tabular numerics on every numeric leading. The bento centerpiece headline uses text-wrap: balance for non-jagged display-tier wraps.

Token-grounded throughout. --surface-elevated, --border, --radius-card, --shadow-tint, every --font-*, every --type-size-*, every --space-*. Zero hardcoded values past the :root seed.
