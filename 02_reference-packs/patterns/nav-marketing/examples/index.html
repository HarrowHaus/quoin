Composition lineage — what nav-marketing consumes

This pattern composes pattern-button-system for every CTA. Optionally consumes pattern-feature-grid in P1 when the mega-dropdown surfaces featured cells.

Consumed primitiveSource packUsed inHow this pattern uses it
<action-button intent="primary"> @quoin/pattern-button-system nav-cta-cluster (every primary CTA) Rightmost CTA in every variant. Inherits all 4 sizes + 8 microstates.
<action-button intent="ghost"> @quoin/pattern-button-system nav-cta-cluster Log-in link · nav-mobile-toggle Ghost-intent for the de-escalated 'Log in' and for the icon-only mobile hamburger button.
<action-button intent="secondary"> @quoin/pattern-button-system nav-cta-cluster (dual-cta / extended registers) Mid-intent CTA for pricing-link / talk-to-sales when alongside a primary CTA.
canonical tokens @quoin/tokens-baseline every CSS value Surfaces / typography / borders / motion / shadows. Nav aesthetic flips with aesthetic-pack swaps; structure invariant.

Accessibility checklist

  • Banner landmark: <header role="banner"> — only one per page.
  • Primary nav landmark: <nav aria-label="Primary"> distinguishes from footer nav and in-page sub-nav. The label is stable: always "Primary".
  • Skip link: First focusable element should be <a class="skip-link" href="#main">Skip to main content</a>. Visually hidden until focused.
  • Current page: Active route gets aria-current="page" + visible underline indicator.
  • Dropdown semantics: Trigger is <button aria-expanded> + aria-controls + aria-haspopup. Panel has aria-labelledby back to trigger. Escape closes + returns focus to trigger.
  • Mobile overlay: role="dialog" + aria-modal="true". Focus trapped while open. Escape and backdrop click close. Body scroll locked.
  • Mobile toggle: aria-expanded + aria-controls + state-paired aria-label ("Open menu" → "Close menu").
  • Announcement region: <div role="region" aria-label="Announcement">. If content updates dynamically, add aria-live="polite".
  • External link affordance: External links carry rel="noopener noreferrer" + optional ↗ glyph.
  • Focus styles: Every interactive element receives visible :focus-visible outline via --focus-ring tokens.
  • Reduced motion: Scroll-state transition and overlay slide-in collapse to instant when prefers-reduced-motion: reduce.
