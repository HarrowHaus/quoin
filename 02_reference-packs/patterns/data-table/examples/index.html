pattern-data-table — specimen

Sortable, filterable, selectable data table. The universal admin / dashboard / list primitive. Eight primitives, three densities, three visual registers, seven pattern states. Composes @quoin/pattern-form-fields for search + filters + selection checkboxes and @quoin/pattern-button-system for sort triggers + pagination + bulk actions.

Composition lineage — what data-table consumes

Consumed primitive Source pack Used in How this pattern uses it
<form-control type="search"> @quoin/pattern-form-fields data-table-toolbar Search input with leading magnifier icon. Inherits 33 tokens + leading/trailing slot system. Debounced input fires filter; row count change announced via aria-live region.
<form-control type="checkbox"> @quoin/pattern-form-fields data-table-row · header master checkbox Selection checkbox per row plus a master checkbox in the header that supports indeterminate state when some-but-not-all rows are selected.
<form-control type="select"> @quoin/pattern-form-fields data-table-pagination (extended register) Rows-per-page selector. Inherits select-input styling + ARIA.
<action-button intent="ghost"> @quoin/pattern-button-system sort triggers · pagination prev/next · row actions · bulk actions (most) The dominant button intent for in-table actions. Inherits all 8 microstates including aria-busy for loading and aria-disabled for unreachable pages.
<action-button intent="critical"> @quoin/pattern-button-system data-table-bulk-actions (Delete) · data-table-empty-state error variant Retry Destructive bulk operations. Must open a modal-dialog confirmation; never executes on first click (delegated to pattern-modal-dialog).
<action-button intent="primary"> @quoin/pattern-button-system data-table-empty-state empty variant CTA Primary CTA for the first-run empty state.
<avatar-stack> primitive @quoin/vocab-app-shell 0.2.0 data-table-cell content-type="avatar-stack" For "Shared with" / "Members" columns. 3-5 avatars with +N overflow indicator.
canonical tokens @quoin/tokens-baseline every CSS value Surfaces / typography / borders / motion / shadows / focus-ring. Table aesthetic flips with aesthetic-pack swaps; structure is invariant.

Accessibility checklist

  • Region landmark: <div role="region" aria-label="<table-name> table"> gives the table surface a named landmark.
  • True table semantics: Always <table> + <thead> / <tbody> / <tfoot>. WCAG 1.3.1 (Info & Relationships) requires real table elements for tabular data — never divs.
  • Column headers: Every <th> in <thead> has scope="col".
  • Sort semantics: Sortable <th> carries aria-sort="none|ascending|descending"; the sort affordance is a <button> with an aria-label that names current state and next action ("Service, sorted ascending. Activate to sort descending.").
  • Selection state: aria-selected="true|false" on <tr> for multi-select. The selection checkbox is the activation surface; aria-selected is the state.
  • Master checkbox indeterminate: When some-but-not-all rows are selected, the master checkbox shows the indeterminate state via the indeterminate property (set via JS, not an HTML attribute).
  • Pagination nav: <nav aria-label="Pagination">; active page button is aria-current="page"; disabled prev/next on edge pages get aria-disabled="true" (kept in DOM for stable layout).
  • Live-region announcements: Pagination "Showing X–Y of Z" is in aria-live="polite"; bulk-actions selection count is also in aria-live="polite"; row-count change from filter updates is announced.
  • Empty-state error: The error-variant heading wraps in role="alert" so AT announces immediately on load failure.
  • Toolbar role: <div role="toolbar" aria-label="Table controls"> — tells AT this is a related-controls cluster.
  • Bulk-actions group: <div role="group" aria-label="Bulk actions for <n> selected rows"> — the count in the label gives context.
  • Destructive confirmation: Critical-intent bulk actions (Delete, Cancel subscriptions) must open a modal-dialog confirmation — never execute on first click.
  • Tabular numerics: Number cells use font-variant-numeric: tabular-nums for column-aligned digits. Right-aligned by default.
  • Status badges: Always carry readable text content; colour is reinforcement, not the only signal (WCAG 1.4.1).
  • Focus styles: Every interactive element receives visible :focus-visible outline via --focus-ring tokens.
  • Reduced motion: Skeleton shimmer animation collapses to none under prefers-reduced-motion: reduce.
