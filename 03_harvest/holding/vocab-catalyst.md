# vocab-catalyst — HELD

**Status:** Held pending license clarification.

## Source

Catalyst — a UI kit sold by Tailwind Labs as part of the paid Tailwind UI
product (now Tailwind Plus).

- Source URL: https://catalyst.tailwindui.com/
- License: **proprietary, paid distribution.** Source code and compiled
  CSS may not be redistributed. The public demo pages may be inspected
  for visual style and patterns, but the underlying component code is
  not under an open license.

## What would need to change before this pack ships

The Phase 3 harvest charter (`03_harvest/README.md` §"Slop avoidance"
and §"License and attribution constraints") forbids harvesting from
sources whose license is incompatible with redistribution. Catalyst's
license is not.

Two paths could change this:

1. **Tailwind Labs publishes a Catalyst pattern dictionary under MIT or
   similar.** The semantic shape (button + button-group + dialog +
   listbox patterns) would then be lawful to harvest.
2. **Pattern-only harvest with explicit Tailwind Labs permission.** The
   public demo pages document the patterns at a high level; if an
   operator obtains explicit written permission from Tailwind Labs to
   ship a semantic-only translation under MIT, that would satisfy the
   charter.

Neither path is open at the time of this harvest, so the pack is
omitted from `03_harvest/packs/` and this note records the reasoning.

## Why hold rather than silently drop

Operators evaluating the harvest set may notice Catalyst missing from
the priority-one vocabulary list. This note explains the absence,
documents the slop-avoidance enforcement, and provides a recovery path
if licensing changes.
