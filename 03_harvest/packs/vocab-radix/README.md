# @quoin/vocab-radix

Radix UI-flavoured accessibility primitives — toggle, switch, slider, progress bar, dialog overlay.

## Primitives

| Primitive | Role |
|-----------|------|
| `<accessible-toggle>` | Two-state toggle button. ARIA-aware. |
| `<switch-toggle>` | On/off switch — a styled checkbox with thumb track visual. |
| `<slider-cell>` | Single-value range slider. |
| `<progress-meter>` | Determinate or indeterminate progress indicator. |
| `<dialog-overlay>` | Modal dialog with backdrop. Native HTML dialog under the hood. |
