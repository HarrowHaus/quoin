# @quoin/vocab-marketing

Marketing-page primitives — hero, feature grid, pricing tier, testimonial, CTA band, FAQ disclosure.

## Primitives

| Primitive | Role |
|-----------|------|
| `<hero-banner>` | Top-of-page marketing banner. Headline + sub-copy + primary action. |
| `<feature-grid>` | Auto-layout grid of feature cards. Wraps responsively. |
| `<feature-cell>` | Single feature unit inside a feature grid. |
| `<pricing-tier>` | Single pricing plan with price, description, feature list, CTA. |
| `<testimonial-quote>` | Customer/user testimonial quote with attribution line. |
| `<cta-band>` | Full-width call-to-action band, typically near page foot. |
| `<faq-disclosure>` | Question/answer pair using a native disclosure. |
