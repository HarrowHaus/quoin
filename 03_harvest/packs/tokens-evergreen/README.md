# @quoin/tokens-evergreen

Segment Evergreen palette translated onto the Quoin canonical semantic-token namespace.

- **Source system:** Evergreen
- **Source organisation:** Segment (Twilio)

- **Source license:** MIT
- **Harvested:** 2026-05-17
- **Notes:** Segment Evergreen default theme colours. The library is in maintenance mode; values are stable. Extraction method: A. Mapping: Evergreen ships scale + intent objects (gray, blue, red, orange, green, etc.) with explicit step keys.

## Mapping

Evergreen has tints rather than a numbered scale; mapped to Quoin's surface ramp. Default brand colour is a muted blue; status colours follow conventional traffic-light pattern.
