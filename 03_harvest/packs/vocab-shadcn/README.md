# @quoin/vocab-shadcn

shadcn/ui-flavoured primitives — command palette, scroll pane, hover card, popover, sheet overlay, tooltip.

## Primitives

| Primitive | Role |
|-----------|------|
| `<command-menu>` | Keyboard-first command palette (Cmd-K style). |
| `<scroll-pane>` | Custom-styled scrollable region. |
| `<hover-card>` | Card that appears on hover, typically for previews. |
| `<popover-panel>` | Click-triggered popover anchored to a button. |
| `<sheet-overlay>` | Drawer / slide-in sheet from an edge of the viewport. |
| `<tooltip-tip>` | Small explanatory tooltip on hover/focus. |
