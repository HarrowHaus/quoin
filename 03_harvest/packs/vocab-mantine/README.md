# @quoin/vocab-mantine

Mantine-flavoured primitives — skeleton placeholder, ring progress, loader spinner, notification toast.

## Primitives

| Primitive | Role |
|-----------|------|
| `<skeleton-placeholder>` | Animated placeholder while content loads. |
| `<ring-progress>` | Circular progress indicator. Inner SVG/canvas supplied by author. |
| `<loader-spinner>` | Indeterminate spinner. |
| `<notification-toast>` | Floating notification toast. |
