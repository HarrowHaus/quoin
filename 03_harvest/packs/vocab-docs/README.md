# @quoin/vocab-docs

Documentation primitives — code fences, callouts, command shells, kbd keys, version badges, table of contents.

## Primitives

| Primitive | Role |
|-----------|------|
| `<code-fence>` | Fenced code block with optional language label and copy affordance. |
| `<callout-box>` | Inline doc callout — note, tip, warning, danger. |
| `<command-shell>` | Terminal/shell command sample with prompt prefix. |
| `<kbd-key>` | Inline keyboard key label. |
| `<version-badge>` | Compact version/status badge, often beside a heading. |
| `<toc-list>` | Table of contents — nested anchor list scaffolding. |
