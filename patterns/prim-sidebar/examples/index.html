Main content

This area takes the remaining inline size beside the sidebar. When the container drops below the 60em threshold (drag the resize handle), the layout switches to stacked.

The switch happens via @container (inline-size >= 60em), not @media — so this primitive is portable into any wrapper layout without re-tuning breakpoints to viewport width.
