data-axis="horizontal" data-max-width="narrow" — readable prose column

This is a centered readable column constrained to a 45ch measure. The container collapses around the natural width of the prose. Operators reach for this when they want a single-column long-form layout: a blog post body, a docs page, a Tufte-style essay.

Prose lives well at 45–75 characters per line; narrower than that and the eye is constantly reversing, wider and lines lose registration.

data-axis="horizontal" data-max-width="wide" — long-form

Wide variant centers an 80ch column. Useful for content surfaces that include figures, code blocks, or multi-column callouts where the 66ch standard is too tight.

Composition — center wrapping a stack

A composed hero stack

center-container provides the max-width + auto-margin centering. An inner stack-container provides the vertical rhythm. Patterns compose these two together to build prose containers, modal contents, hero blocks, and empty states.
