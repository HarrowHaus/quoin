The six semantic registers. Each maps to a token-driven colour + an ARIA label per role.

note

Note

The neutral register. Use for context that's worth surfacing but isn't strictly required reading.

tip

Tip

Optional improvement or shortcut. Doesn't change the outcome if ignored.

warning

Warning

Something that may cause unexpected behavior if not handled. Read before proceeding.

danger

Danger

Destructive or irreversible action. The operator can elevate role='alert' here if the AT announcement should be more prominent.

success

Success

Confirms a positive outcome. Common in step-by-step docs ("you're done — your environment is ready").

info

Heads up

Informational context. Similar to note but with a stronger visual signal.

Inline register example

Inline register has no surface fill and no rounded corners — just a vertical accent rule. Fits smaller call-outs without visual weight.
