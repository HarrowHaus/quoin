Quickstart

This is a stand-in article body. Track B will ship @quoin/pattern-prose-body with full prose semantics (measure, leading, link/list/quote styling, dropcap variant).

Install

Install the Quoin compiler and the patterns + aesthetic packs you want to consume:

npm install @quoin/compiler @quoin/tokens-baseline \
  @quoin/aesthetic-default \
  @quoin/pattern-nav @quoin/pattern-hero

Tip

Patterns + aesthetics are independent. Start with one of each and add more as your design surface grows.

Configure

Create a quoin.config.js declaring which packs to compose:

export default {
  tokens: '@quoin/tokens-baseline',
  aesthetic: '@quoin/aesthetic-default',
  patterns: ['@quoin/pattern-nav', '@quoin/pattern-hero', '@quoin/pattern-footer-mega']
};

Heads up

The aesthetic pack must declare values for every token referenced by your patterns. Mismatch will fail the build.

Ship

Run the build and deploy the output. Quoin's compile output is plain CSS + HTML — there is no runtime.

npx quoin build
npx wrangler deploy   # or your platform's deploy command

Next steps

Read the specification for the full pack format. Browse the aesthetic catalog for visual options. Open the playground to swap patterns and aesthetics live.
