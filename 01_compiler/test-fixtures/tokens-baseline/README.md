# @quoin/tokens-baseline (Phase 1 fixture)

A minimal neutral token pack used by the Phase 1 compiler test suite.
This fixture is not the canonical Phase 2 deliverable — it exists so that
the compiler can exercise the full pack-loader + token-resolver path
against a complete, schema-conformant token pack.

Implements every name in the canonical semantic-token namespace defined
in `00_spec/tokens.md` §2 against a stone-greyscale base palette.
